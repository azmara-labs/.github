# azmara-labs/.github

Default community health files for every `azmara-labs` repository. GitHub
applies these automatically to any repo in the org that doesn't define its
own copy — nothing needs to be duplicated per-repo.

Currently provides:
- `PULL_REQUEST_TEMPLATE.md` — default PR checklist (tests, types, lint, changeset, CLA)

A repo-level file of the same name (e.g. `azmara-platform/.github/PULL_REQUEST_TEMPLATE.md`)
always takes precedence over this default if one exists.
