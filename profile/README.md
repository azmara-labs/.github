# Azmara Labs

Azmara Labs (Azmara Technologies) is a New Zealand-based software and security engineering studio, building a data-first application platform and the tooling that keeps it reliable.

🌐 [azmara.io](https://azmara.io) · 📍 Hamilton, NZ

## What we're building

| Project | Description |
| --- | --- |
| [**azmara-platform**](https://github.com/azmara-labs/azmara-platform) | Modern data-first application runtime — a monorepo covering core, db, query, ui, ai, security, and cli packages. |
| [**pr-quality-checks**](https://github.com/azmara-labs/pr-quality-checks) | Reusable GitHub Actions workflows enforcing package hygiene, publish safety, and config standards across our repos. |
| [**mui-carousel**](https://github.com/azmara-labs/mui-carousel) | A carousel component for MUI. |

## How we work

- Small, verified changes over big-bang merges — every PR runs through [`pr-quality-checks`](https://github.com/azmara-labs/pr-quality-checks) before it lands.
- Dependencies are kept current and audited; security advisories get fixed, not silenced.
- Open source where it makes sense — issues and PRs are welcome on our public repos.
